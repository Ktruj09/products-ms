# Product Microservice 

## Project setup

```bash
$ npm install
```


## Settings for building the project

```bash

$ Create a .env file in the root directory
$ Example .env.template 
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
